Table in Html 5

In HTML 5 you can create a table using the table tag

For example:


My favourite youtuber Descriptions
Pewdiepie Real name is Felix Kjiellberg, is the most subscribed individual youtuber. Funny and calm. He was like a father to me
videogamedunkey Is one of the best comedian on the website. Also a very good game reviewer.
Casually explained Top notch sarcasm. My dream is one day be able to tell such jokes with such a straight voice
My favourite youtuber table

Here table tag is used to make a table. Between that tag is a table. That how the browser understand our code. After that, we've got the (tr) and (td) tag. For each (tr) tag there's gonna be a new row, and the (td) tag defines a new table data, which in this case is a new column. We can separate the table into parts: table head (thead), table body (tbody), and table foot (tfoot). table head is for the header of the table, which in my table is "My favourite youtuber" and "Descriptions". Inside (thead) we would continue to declare (tr) for new table head row and (th) for the header of certain table column. (tfoot) is basically for things like sum up, conclusions, etc.. Table body (tbody) is mainly to separate the head, body, and the foot.

basically we have this structure

  • table
    • thead
      • tr
        • th__/th
        • th__/th
      • /tr
    • /thead
    • tbody
      • tr
        • td__/td
        • td__/td
      • /tr
    • /tbody
    • tfoot
    • /tfoot
  • /table

We actually can use td, th, tr for almost every task we need to do with our table. Just that using the right one for the right situation will help the browser understand our code better. Moreover, some tag like th is actually meant to be used as a header tag for table.

...And also, we can use other extra tag like colgroup (for defining a group of column in a table) and caption (for specifying a title of a table). You can find more detailed information about it here
