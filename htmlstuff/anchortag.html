Anchor tag, or how you refer to a hyperlink in your html document.

...by yours truly

An anchor tag, or an (a) tag is how you include a hyperlink to another webpage in your own sites.


Its syntax goes as this

(a href="enter link here")the text you want to display(/a)

Please note that you can only use this tag with href attribute. It contains the link to websites.
